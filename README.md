# Task Manager Application

## Backend
- Java Spring Boot, H2 Database
- CRUD operations for Tasks

## Frontend
- Angular, Bootstrap
- Display and manage tasks via forms

### Running the App
1. **Backend:**
   ```bash
   cd backend
   mvn spring-boot:run

2. **Frontend:**
   ```bash
   cd frontend
   npm install
   npm start
